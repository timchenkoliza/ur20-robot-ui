# ur20-robot-ui

Functional prototype for UR20 robot swarm management interface.

**Features:**
- Dashboard view
- Operators mode (simple Start/Stop)
- Developers mode (AI debug, parameters)
- Demo mode (visualization for customers)
- Settings

## Live Demo
https://timchenkoliza.github.io/ur20-robot-ui/

## How to use
1. Open `index.html` in browser
2. Use navigation tabs to switch modes
3. All buttons are interactive

## Modes
- **Dashboard**: System overview
- **Operators**: Simplified control
- **Developers**: Full debug info
- **Demo**: Customer presentation mode
